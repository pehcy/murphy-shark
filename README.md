# 11ty blog for Murphy Shark

This blog was created with Eleventy 2.0 (stable), and been host in GitHub page. 
The previous site has been depreciated (The Gatsby site) and has been migrated 
to use 11ty.

## Local Development Setup

Prerequisites: Node 12+, NVM

## Local Development Workflow

1. Type `npm install` to install all dependencies.

2. Once installed and `node_modules` updated, type `npm run start` in the terminal.

3. Navigate to `http://localhost:8080/` and see the webpage.

To install new local npm packages, use `npm install PACKAGE_NAME --save-dev`.
